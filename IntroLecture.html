Basic Web Schematic

Main Web Protocols

The Internet is built on a number of communications protocols.

The main ones are:

  • Hypertext Transfer Protocol (HTTP)
  • Hypertext Transfer Protocol Secure (HTTPS)
  • File Transfer Protocol (FTP)
  • Multi-Purpose Internet Mail Extension (MIME)
  • Simple Main Tranfer Protocol (SMTP)
  • Post Office Protocol (POP3)
  • Internet Mail Access Protocol (IMAP)

TCP/IP

  • Transmission Control Protocol (TCP)
  • Internet Protocol (IP)

TCP and IP serve different purposes. TCP is concerned with making sure data packets are transmitted and received correctly. IP is concerned with the address the packets go to.
